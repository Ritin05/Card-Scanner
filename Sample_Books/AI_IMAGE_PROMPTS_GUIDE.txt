╔══════════════════════════════════════════════════╗
║  TIPS FOR BEST RESULTS                           ║
╚══════════════════════════════════════════════════╝

1. CONSISTENCY: Add this to EVERY prompt for consistent style:
   "3D cartoon Pixar style, children's book illustration, vibrant colors"

2. BING IMAGE CREATOR (Recommended for FREE):
   - Go to: https://www.bing.com/images/create
   - Paste the prompt
   - Download the best image (1024x1024)
   - Use in your book

3. IMAGE SIZE for Amazon KDP:
   - Book size: 8.5" x 8.5" (square) or 8" x 10"
   - Image resolution: at least 300 DPI
   - Generate at highest resolution possible

4. EDITING IMAGES:
   - Use Canva (free) to resize/crop images
   - Remove any text watermarks if present
   - Ensure consistent color tone across all pages

5. STYLE KEYWORDS to add if images look different:
   - "same style as previous"
   - "consistent character design"
   - "cute chibi anime style" (alternative style)
   - "soft pastel colors" (for gentler look)

6. CHARACTER CONSISTENCY:
   - Krishna: Always blue skin, peacock feather, golden jewelry
   - Devaki: Red sari, bindi, long black hair
   - Vasudev: Yellow/white clothes, turban
   - Kansa: Dark clothes, golden crown, angry face
   - Yashoda: Colorful sari, kind face, bindi
